# NotePad应用操作流程

## 1. 主界面

**截图**：JZ5C6}@4G]WK%B2{_{[)@13.png

**界面说明**：
- 顶部自定义ActionBar，显示应用名称
- 笔记列表以卡片形式展示，包含标题、修改时间和标签
- 右上角有添加新笔记的"+"图标
- 支持左右滑动手势隐藏/显示列表

## 2. 进入新建笔记界面

**截图**：)P1E_KJK0CC`W8VC@FOMOE0.png

**操作步骤**：
1. 点击主界面右上角的"+"图标
2. 系统自动进入新建笔记编辑界面
3. 界面显示空白的标题和正文区域
4. 顶部ActionBar显示"保存"和"返回"按钮

## 3. 输入笔记内容

**截图**：DQ]VL11US40[W)42~(L1@34.png

**操作步骤**：
1. 在正文区域输入笔记内容
2. 系统自动保存输入内容
3. 标题区域显示默认的"无标题"或自动提取的内容

## 4. 添加标签

**截图**：P3AT~{G9O2$D5MKR]75B6LP.png

**操作步骤**：
1. 点击标签按钮（通常位于编辑界面底部或菜单中）
2. 弹出标签选择对话框
3. 选择已有标签或点击"新建标签"
4. 输入新标签名称并确认
5. 标签添加成功，显示在笔记中

## 5. 修改笔记标题

**截图**：E5YMSSCLN~FP~2XC@XX(68Y.png

**操作步骤**：
1. 点击笔记标题区域
2. 标题进入编辑状态
3. 输入新的标题
4. 点击其他区域或保存按钮完成修改

## 6. 保存笔记

**截图**：V2~{{}P`_7LFW$A`97YA[C9.png

**操作步骤**：
1. 点击顶部ActionBar的"保存"按钮
2. 系统保存笔记内容到数据库
3. 自动返回主界面
4. 新创建的笔记显示在列表顶部

## 完整操作流程总结

1. 打开NotePad应用，进入主界面
2. 点击右上角"+"图标，进入新建笔记界面
3. 在正文区域输入笔记内容
4. 添加适当的标签
5. 修改笔记标题
6. 点击保存按钮，完成笔记创建

## 注意事项

- 应用支持自动保存，即使未点击保存按钮，内容也会定期自动保存
- 笔记保存后会显示在主界面列表中
- 可以通过长按笔记进行复制或删除操作
- 支持通过标签过滤和搜索功能快速查找笔记

## 快捷键与手势

- 左滑：隐藏笔记列表，显示背景
- 右滑：显示笔记列表
- 长按笔记：显示上下文菜单
- 点击标签：过滤显示对应标签的笔记
