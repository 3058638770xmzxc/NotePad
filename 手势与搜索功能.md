# NotePad应用手势与搜索功能

## 1. 左右滑动手势

### 1.1 主界面

**截图**：(5Y60A~V~C~EZQS@CPTX02C.png

**界面说明**：
- 显示完整的笔记列表
- 顶部ActionBar显示应用名称
- 笔记以卡片形式排列

### 1.2 向左滑动显示空背景

**截图**：JZ5C6}@4G]WK%B2{_{[)@13.png

**操作步骤**：
1. 在主界面上从右向左滑动
2. 笔记列表会向左滑出屏幕
3. 只显示应用背景（颜色或图片）
4. 适合查看完整背景效果

### 1.3 向右滑动恢复原状

**操作步骤**：
1. 当列表隐藏时，从左向右滑动
2. 笔记列表会平滑滑入屏幕
3. 恢复到正常显示状态

## 2. 标签点击过滤功能

**截图**：GM~[J1DVN$B[8GC]HR@I`IS.png

**操作步骤**：
1. 在主界面点击任意笔记的标签
2. 系统自动过滤显示该标签下的所有笔记
3. 顶部ActionBar标题更新为"应用名称 - 标签名"
4. 列表只显示包含该标签的笔记

**特点**：
- 支持多标签管理
- 标签过滤状态持久化
- 可通过"清除过滤"按钮恢复显示所有笔记

## 3. 标签内搜索功能

**截图**：6D)_SCZD4DYOK$`%3~@]X]0.png

**操作步骤**：
1. 先点击标签进入标签过滤状态
2. 点击搜索图标
3. 输入搜索关键词"周二"
4. 系统在当前标签过滤结果中进行搜索
5. 只显示同时匹配标签和搜索关键词的笔记

**特点**：
- 搜索范围限定在当前标签内
- 搜索结果实时更新
- 标题显示"应用名称 - 标签名 (搜索词)"

## 4. 全局搜索功能

**截图**：%8E}Y07FFQ$GOSE81_KP{1G.png

**操作步骤**：
1. 在主界面点击搜索图标（无标签过滤状态）
2. 输入搜索关键词
3. 系统在所有笔记中进行搜索
4. 显示所有匹配搜索关键词的笔记

**特点**：
- 搜索范围覆盖所有笔记
- 支持标题、内容和标签搜索
- 搜索结果实时更新
- 标题显示"应用名称 - 搜索词"

## 功能对比

| 功能 | 操作方式 | 搜索范围 | 显示效果 |
|------|----------|----------|----------|
| 全局搜索 | 直接点击搜索图标 | 所有笔记 | 显示所有匹配笔记 |
| 标签内搜索 | 先点击标签，再搜索 | 当前标签内 | 只显示标签内匹配笔记 |

## 操作提示

1. **手势操作**：
   - 滑动时保持手指与屏幕接触
   - 滑动距离建议超过50像素
   - 支持快速和慢速滑动，效果一致

2. **搜索功能**：
   - 支持部分匹配搜索
   - 支持中文和英文搜索
   - 搜索结果按修改时间排序

3. **标签功能**：
   - 每个笔记只能设置一个标签
   - 支持创建新标签
   - 点击标签名即可过滤

## 应用场景

1. **查看背景**：向左滑动隐藏列表，查看完整背景图片或颜色
2. **快速筛选**：点击标签快速查看相关笔记
3. **精准搜索**：先筛选标签，再搜索，提高搜索准确性
4. **全局查找**：直接搜索，查找所有相关笔记

| 操作 | 手势/快捷键 | 效果 |
|------|-------------|------|
| 隐藏列表 | 向左滑动 | 显示空背景 |
| 显示列表 | 向右滑动 | 恢复笔记列表 |
| 搜索 | 点击搜索图标 | 打开搜索界面 |
| 过滤标签 | 点击标签 | 显示标签内笔记 |
