# NotePad应用设置功能

## 1. 进入设置界面

**截图**：SOZQ{M`1@SY11YM{64X0SBW.png

**操作步骤**：
1. 在主界面点击菜单图标
2. 从弹出的菜单中选择"设置"选项
3. 进入设置界面

**界面说明**：
- 显示各种设置选项
- 支持背景设置和排序设置

## 2. 背景颜色选择

**截图**：BVA`{3NU8Q4IK91)[ZXND4O.png

**操作步骤**：
1. 在设置界面点击"背景颜色"选项
2. 弹出颜色选择器
3. 显示可选的颜色选项（白色、米色、蓝色、绿色等）
4. 点击选择喜欢的颜色
5. 系统自动应用所选颜色

**可选颜色**：
- 白色：简洁清爽
- 米色：温暖舒适
- 蓝色：冷静专业
- 绿色：自然放松

## 3. 背景图片裁剪

**截图**：FT%FS%0(%JJ)04CSBA{J@WU.png

**操作步骤**：
1. 在设置界面点击"背景图片"选项
2. 从相册选择图片
3. 进入图片裁剪界面
4. 调整裁剪框大小和位置
5. 选择合适的裁剪比例
6. 点击"确定"完成裁剪

**裁剪选项**：
- 自由裁剪：手动调整裁剪区域
- 固定比例：支持多种预设比例
- 预览功能：实时查看裁剪效果

## 4. 排序设置

**截图**：U5_H[%9M(%M}(R3Y7O{BJYQ.png

**操作步骤**：
1. 在设置界面点击"排序方式"选项
2. 显示排序选项列表
3. 选择排序方式（修改时间升序/降序）
4. 系统自动应用新的排序方式

**可选排序方式**：
- 修改时间升序：最早修改的笔记在前
- 修改时间降序：最近修改的笔记在前

## 5. 编辑器设置

**截图**：3{W%IUUD35`_{A~XM%F2ZWG.png

**操作步骤**：
1. 在设置界面点击"编辑器设置"选项
2. 进入编辑器设置界面
3. 调整编辑器相关参数
4. 保存设置

**编辑器设置选项**：
- 字体大小调整
- 行高设置
- 自动保存间隔
- 拼写检查开关

## 7. 设置应用效果

**截图**：BVA`{3NU8Q4IK91)[ZXND4O.png

**操作效果**：
1. 设置变更实时生效
2. 无需重启应用

4. 支持恢复默认设置

## 完整设置流程

1. 进入主界面
2. 打开菜单，选择设置
3. 根据需要调整各项设置
4. 设置自动保存，实时生效
5. 返回主界面查看效果

## 设置功能特点

1. **直观易用**：简洁的设置界面，操作简单
2. **实时生效**：设置变更立即应用，无需重启
3. **个性化定制**：丰富的定制选项，满足不同用户需求
4. **自动保存**：设置自动保存到本地，下次启动自动加载
5. **分类清晰**：设置项分类明确，便于查找和调整
